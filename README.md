# Agent Hub

[![Latest Release](https://img.shields.io/github/v/release/YOOJUNGYU/agent-hub)](https://github.com/YOOJUNGYU/agent-hub/releases/latest)

**한국어** | [English](README.en.md)

**내 PC의 AI 에이전트(Claude)를, 자리를 비워도 휴대폰에서 지켜보고 제어하는 프로그램입니다.**

> ### 🔐 안전한 로컬 네트워크 안에서, 내가 승인한 기기만
> 자리를 비워도 같은 네트워크의 휴대폰으로 Claude의 **질문에 답하고, 위험한 작업을 승인하고, 명령까지** 보냅니다.
> **제어·모니터링 통신은 집·회사 네트워크(LAN)** 안에서만 오가고(앱을 닫았을 때 오는 알림만 암호화되어 외부 푸시 서비스를 거칩니다), **어떤 기기가 접근·제어할지는 내가 PC에서 직접 승인·관리**합니다.

---

## 📖 사용 가이드

설치 → 인증서 등록 → 앱 설치(홈 화면 추가) → 알림 켜기까지, **그림으로 한눈에 따라 하는 가이드**를 준비했습니다.

### 👉 **[사용 가이드 열기](https://yoojungyu.github.io/agent-hub/)**

## 다운로드 & 설치

1. **[최신 버전 다운로드](https://github.com/YOOJUNGYU/agent-hub/releases/latest)** 페이지에서 **`AgentHub-win-Setup.exe`**를 받습니다.
2. 받은 파일을 **더블클릭**해 설치합니다. (관리자 권한 필요 없음)
3. 처음 실행할 때 인증서 설치 창이 뜨면 **예**를 눌러 주세요.

> 💡 파란 화면으로 **"Windows의 PC 보호"** 경고가 뜨면 **추가 정보 → 실행**을 누르면 됩니다. (서명되지 않은 프로그램이라 나타나는 안내입니다.)

설치가 끝나면 프로그램이 오른쪽 아래 **작업 표시줄 트레이**에 상주합니다.

## 빠른 시작 (휴대폰 연결)

1. PC 콘솔 상단의 **접속 주소**를 확인합니다. (예: `https://192.168.0.10:47600`)
2. 휴대폰을 PC와 **같은 네트워크**에 두고, 그 주소를 브라우저에 입력합니다.
3. **기기 인증 요청** → PC 콘솔에서 **승인**하면 모니터 화면이 표시됩니다.

> 📱 **알림**을 받으려면 **인증서 등록 + 홈 화면에 추가(앱 설치)**가 필요합니다. 그림으로 따라 하는 자세한 과정은 **[사용 가이드](https://yoojungyu.github.io/agent-hub/)**를 보세요.

## 주요 기능

- **실시간 모니터링** — 세션 상태(작업 중/대기/종료)·현재 작업·경과 시간·누적 토큰을 실시간 표시.
- **폰에서 원격 제어** — 질문 답변·위험 작업 허용/거부·명령 전송. (선택: 세션 터미널로 프롬프트·/슬래시 명령까지)
- **앱을 닫아도 알림** — 세션이 당신을 기다릴 때 푸시가 오고, Claude의 **마지막 메시지**가 본문에 표시됩니다.
- **기기 인증** — 승인된 기기만 접근. PC에서 요청을 승인·해제·삭제할 수 있습니다.
- **트레이 상주 · 자동 업데이트** — 평소엔 조용히 트레이에 있고, 새 버전은 자동으로 적용됩니다.

## Claude Code 공식 원격 제어와 무엇이 다른가요?

Claude Code에는 세션을 원격에서 이어받는 공식 **Remote Control** 기능이 있습니다. 목표(자리를 비운 사이 폰에서 Claude를 제어)는 비슷하지만, Agent Hub는 **연결 방식과 전제가 근본적으로 다릅니다.**

| | **Claude Code Remote Control (공식)** | **Agent Hub** |
|---|---|---|
| **연결 경로** | Anthropic 클라우드(API) 경유 | **내 LAN 안 로컬 서버만** — 제어·모니터링 통신이 외부로 안 나감 |
| **접근 범위** | 인터넷 어디서나 | **같은 네트워크(집·회사 LAN)** 안에서 |
| **필요 조건** | Pro/Max/Team/Enterprise 구독 + 계정 로그인 | **구독 무관** — 로컬에 Claude Code만 있으면 됨 |
| **접근 통제** | 계정·조직 정책 / 신뢰 기기 | **내 PC에서 기기별 직접 승인** |
| **대상 세션** | 원격 제어로 켠 세션 | 로컬에서 돌고 있는 **세션들을 자동 감지**해 한눈에 |
| **클라이언트** | 네이티브 모바일 앱 · 웹 | **PWA(홈 화면 앱)** + PC 콘솔 |

> 💡 한마디로, **클라우드를 거치지 않고 내 네트워크 안에서, 내가 승인한 기기만** 쓰고 싶을 때 Agent Hub가 맞습니다. (앱을 닫았을 때 오는 알림만 암호화되어 외부 푸시 서비스를 거칩니다.)

## 설정 · 업데이트

- 콘솔 **설정** 탭에서 서버 **포트**(기본 `47600`)와 **표시 언어**(한국어/English)를 바꿀 수 있습니다.
- 새 버전은 자동으로 내려받아 **다음 실행 때 적용**됩니다. (트레이 우클릭 → *지금 업데이트 후 재시작*으로 즉시 적용도 가능)
